Selamat Datang di Belajar HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web.

Elemen Dasar dalam HTML:

  • Heading (Judul)
  • Paragraph (Paragraf)
  • Link (Tautan)
  • Gambar
  • Daftar

Hal yang perlu dikuasai dalam Desain Web:

  1. HTML
  2. CSS
  3. JavaScript
  4. Responsive Web Design
  5. Konsep UI/UX
